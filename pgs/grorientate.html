    • Un graf orientat este o pereche ordonată de mulțimi notată G=(V,U) unde V este o mulțime finită şi nevidă ale cărei elemente se numesc noduri sau vârfuri și U este o mulțime de perechi ordonate de elemente distincte din V ale cărei elemente se numesc arce.
    Pentru arcul u=(x,y), se numesc extremități ale sale nodurile x şi y, x este extremitate inițială, predecesor al lui y, y este extremitate finală, succesor al lui x. Dacă într-un graf există arcul u=(x,y) se spune despre nodurile x şi y că sunt adiacente. Dacă u1 și u2 sunt două arce ale aceluiaşi graf, u1 și u2 sunt incidente dacă au o extremitate comună (u1=(x,y) şi u2=(y,z) sunt incidente). Dacă u1=(x,y) este un arc într-un graf, se spune despre el și nodul x sau nodul y că sunt incidente.
    Într-un graf orientat G=(V,U) grad exterior al nodului x este numărul arcelor de forma (x,y), notat d+(x), iar grad interior al nodului x este numărul arcelor de forma (y,x), notat d-(x). Într-un graf orientat, suma gradelor exterioare a tuturor nodurilor este egală cu suma gradelor interioare a tuturor nodurilor și cu numărul de arce. Un nod x se numește izolat dacă d+(x)=d-(x)=0. Într-un graf orientat cu n noduri, în care nu există mai multe arce de la un nod la altul, matricea de adiacență a grafului este o matrice cu n linii și n coloane și elemente 0 sau 1, A(i,j)=1 dacă există arcul (i,j) și A(i,j)=0 dacă există nu arcul (i,j).
