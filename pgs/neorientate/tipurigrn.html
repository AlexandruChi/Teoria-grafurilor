    Un graf neorientat oarecare poate avea mai multe grafuri parțiale și subgrafuri, dar un unic graf complementar
    Numărul de grafuri parțiale ale unui graf G este de 2m, m → numărul de muchii. Numărul de grafuri parțiale ale unui graf G este de 2n-1, n → numărul de vârfuri.
    Un graf parțial al unui graf G=(X,U), graful neorientat G1=(X,U1), unde U1⊆U. Un graf parțial G1 al lui G, este chiar G, sau se obține din G păstrând toate vârfurile si eliminând niște muchii. Un subgraf al grafului G=(X,U), graful neorientat G1=(X1,U1) unde X1⊆X, U1⊆U iar U1 conține toate arcele din U care au extremitățile în X1. Un subgraf al grafului G se obține ștergând eventual anumite vârfuri și odată cu acestea și muchiile care le admit ca extremitate (nu se pot șterge toate vârfurile deoarece s-ar obține un graf cu mulțimea vârfurilor vidă). Un graf complementar al grafului G=(X,U), graful neorientat G1=(X,U1), cu proprietatea că două vârfuri x și y sunt adiacente în G1 dacă și numai dacă nu sunt adiacente în G.

    Graf bipartit complet
    Se pot construi 2n(n-1)⁄2 grafuri neorientate distincte cu n vârfuri.
    Un graf neorientat se numește graf nul dacă mulțimea muchiilor este vidă. Într-un graf nul toate vârfurile sunt izolate. Graful G se numește graf complet dacă oricare două vârfuri distincte ale sale sunt adiacente. Un graf complet cu n vârfuri se notează Kn. Într-un graf complet cu n vârfuri sunt Cn2=n(n-1)⁄2 muchii și fiecare vârf are gradul n-1. Graful regulat este un graf în care toate nodurile au același grad. Un graf G=(X,U) se numește graf bipartit dacă există două mulțimi nevidă A și B astfel încât X=A∪B, A∩B=Ø și orice muchie u a lui G are o extremitate în A iar cealaltă în B. Mulțimile A și B formează o partiție a lui X. Un graf bipartit G=(X,U) se numește bipartit complet dacă pentru oricare două vârfuri x∈A și y∈B, există în graf muchia [x,y]∈U.
    Ciclu hamiltonian: [1,4,2,3,7,6,5,1]
    Ciclu eulerian: [1,4,2,1,3,2,7,3,5,7,6,5,1]
    Un graf hamiltonian este un graf care conține un ciclu hamiltonian. Se numește ciclu hamiltonian un ciclu elementar care conține toate vârfurile grafului. Un graf neorientat G este hamiltonian dacă are n≥3 vârfuri şi gradul oricărui vârf verifică inegalitatea d(x)≥n⁄2. Un graf eulerian este un graf care conține un ciclu eulerian. Se numește ciclu eulerian un ciclu care conține toate muchiile grafului. Un graf G=(X,U), fără vârfuri izolate, este eulerian dacă este conex și gradele tuturor vârfurilor sale sunt numere pare.
